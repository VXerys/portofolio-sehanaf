# public/images

Folder ini menyimpan aset gambar statis untuk konten visual dan identitas portfolio.

## Kegunaan
- Menempatkan gambar yang diakses langsung melalui path publik.
- Menjadi lokasi default aset visual non-dinamis.

## Aturan
- Gunakan penamaan deskriptif dan konsisten.
- Optimalkan ukuran gambar sebelum commit.
- Pisahkan gambar konten utama dari gambar dekoratif bila skala aset bertambah.

## Batasan
- Jangan menyimpan aset sensitif.
- Hindari duplikasi aset dengan nama berbeda.
